# CSS Lesson Notes

## What I have learn :
- CSS stand for Cascading Style Sheets
- CSS Syntax : Selector and Declaration
- How to Add CSS : Inline CSS, Internal CSS and External CSS
- Using !important : to force a CSS rule to override other styles
- Type of CSS Selector : Universal Selector (*), Type/Element Selector, Class Selector (.classname), ID Selector (#idname), Group Selector (selector1, selector2), Combinator Selector, Attribute Selector, Pseudo-class Selector.
- CSS Size Units : there are 2 main type (Absolute units and Relative units)
